# :blue_book: How to get it running

Make sure to install :cd::

- Node
- Node Package Manager (NPM)
- Git

Then to run the project :runner::

- Run `npm i` in the project's root directory
- Run `npm run dev`
- Go to localhost:8000 to see the webapp
